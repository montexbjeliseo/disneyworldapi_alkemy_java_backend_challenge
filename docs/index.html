Registro

El registro se lleva a cabo con nombre de usuario, email y contraseña. Mediante el endpoint /auth/register y el método POST.

Ejemplo:


Se registrara el usuario en base de datos y se devolverá un token para ser usado inmediatamente, que durará 8 horas. Ademaás se le notificará por email.

Listado de personajes, peliculas y generos

Para esto hay que tener un token. Mediante el endpoint /characters y el método GET.

Ejemplo:


Se devolverá una lista de personajes, que tendrá el nombre e imagen.

Filtros:

  • age: filtra por edad
  • name: filtra por nombre
  • movie: filtra por peliculas en las que participen
  • detail: regresa una lista con los personajes y todos sus detalles
  • order: ordena la lista, ejemplo: /characters?order=asc|desc o /characters?order=desc|asc

Todos los filtros pueden ser combinados y siempre a devolver un arreglo.

Peliculas

Endpoint: /movies GET

Generos

Endpoint: /genres GET

Creacion de personajes, peliculas y generos

Para esto hay que tener un token. Mediante el endpoint /characters/create y el método POST.

Ejemplo:


Se devolverá el personaje con todas sus propiedades.

Peliculas

Endpoint: /movies/create POST

Se debe proveer: title, image, rate, creationDate.

Generos

Endpoint: /genres/create POST

Se debe proveer: name, image

Actualizar

Ejemplo:


Se devolverá el personaje con todas sus propiedades.

Endpoint: /movies/update PUT

Generos

Endpoint: /genres/update PUT

Agregar

Se debe incluir obligatoriamente el ID de la pelicula y del personaje Mediante el endpoint /movies/{idMovie}/characters/{idCharacter} y el método POST.

Ejemplo:


Se devolverá la pelicula con todas sus propiedades.

Peliculas

Endpoint: /genres/{idGenre}/movies/{idMovie} POST

Eliminacion

Se debe incluir obligatoriamente el ID de la pelicula, personaje o genero Mediante el endpoint:
Personaje: /characters/{id}/delete y el método DELETE.


Pelicula: /movies/{id}/delete y el método DELETE.


Genreos: /genres/{id}/delete y el método DELETE.

Ejemplo:

Elimnar un personaje de una pelicula

Endpoint: /movies{idMovie}/characters/{idCharacter DELETE

Elimnar una pelicula de un genero

Endpoint: /genres{idGenre}/movies/{idMovie/span> DELETE
